# NexusAnalyst

Это мой второй проект в мире машинного обучения.

### О чём этот проект

Я создал простенького **автономного аналитического агента**, который умеет:

- Работать с моими личными документами (PDF)
- Сам решать, где искать ответ — в моих документах или в интернете
- Критически оценивать свою работу
- Если в документах информации недостаточно — самостоятельно идти в сеть и дополнять ответ
- Вести осмысленный диалог с сохранением контекста

По сути, я построил мини-версию той архитектуры, которую сейчас используют серьёзные RAG-системы.

### Как запустить проект

#### 1. Клонируй репозиторий
```bash
git clone https://github.com/твой_ник/nexus_analyst.git
cd nexus_analyst
```
#### 2. Создай виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate    # Mac/Linux
# venv\Scripts\activate     # Windows
```

#### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

#### 4. Настрой .env файл
Создай в корне проекта файл .env и добавь туда свой ключ:
```env
GIGACHAT_CREDENTIALS=твой_ключ_в_base64_формате
```

#### 5. Подготовь документы (один раз)
Положи свой PDF-файл в папку проекта и запусти (внутри ingest.py укажи путь к своему файлу):
```bash
python ingest.py
```

После этого появится папка faiss_index — это база знаний твоего агента.

#### 6. Запусти агента
```bash
python main.py
```
